How this works

Model Simplified Friis link budget + footprint geometry. Educational — not AST proprietary RF planning or ITU/FCC compliance model.

Direct-to-cell architecture verified concept

Large LEO phased array → beam to handset using MNO-licensed low-band spectrum (700/800 MHz SCS). Ground gateways integrate with partner core networks. DCD · FCC

Block 1 array ~64 m² · 98.9 Mbps peak demo
Block 2 array ~223 m² · ~200 Mbps peak target
Patents 3,900+ claims per IR
Manufacturing ~500k sq ft Midland TX

FCC Handset TX ≤23 dBm typical (Part 22/24). NGSO operators file max EIRP per 47 CFR § 25.114. AST authorized DA 26-391 — model EIRP is illustrative, not a filing value.
